# useConnections

Hook for getting active connections.

## Import

```ts
import { useConnections } from "sn-wolf"
```

## Usage

```tsx [index.tsx]
import { useConnections } from "sn-wolf"

function App() {
  const connections = useConnections()
}
```

## Parameters

```ts
import { type UseConnectionsParameters } from "sn-wolf"
```

### config

`Config | undefined`

[`Config`](/react/api/createConfig#config) to use instead of retrieving from the nearest [`WolfProvider`](/react/api/WolfProvider).

:::code-group
```tsx [index.tsx]
import { useConnections } from "sn-wolf"
import { config } from "./config" // [!code focus]

function App() {
  const connections = useConnections({
    config, // [!code focus]
  })
}
```
