# useConfig

Hook for getting [`Config`](/react/api/createConfig#config) from nearest [`WolfProvider`](/react/api/WolfProvider).

## Import

```ts
import { useConfig } from "sn-wolf"
```

## Usage

```tsx [index.tsx]
import { useConfig } from "sn-wolf"

function App() {
  const config = useConfig()
}
```
