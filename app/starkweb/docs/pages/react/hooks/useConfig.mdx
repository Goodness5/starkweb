# useConfig

Hook for getting [`Config`](/react/api/createConfig#config) from nearest [`StarkwebProvider`](/react/api/StarkwebProvider).

## Import

```ts
import { useConfig } from "starkweb/react"
```

## Usage

```tsx [index.tsx]
import { useConfig } from "starkweb/react"

function App() {
  const config = useConfig()
}
```
