# useSwitchChain

Hook for switching the target chain for a connector or the Wolf [`Config`](/react/api/createConfig#config).

## Import

```ts
import { useSwitchChain } from "starkweb/react"
```

## Usage

```tsx [index.tsx]
import { useSwitchChain } from "starkweb/react"

function App() {
  const { chains, switchChain } = useSwitchChain()

  return (
    <div>
      {chains.map((chain) => (
        <button key={chain.id} onClick={() => switchChain({ chainId: chain.id })}>
          {chain.name}
        </button>
      ))}
    </div>
  )
}
```

:::tip
When connected, `switchChain` will switch the target chain for the connector. When not connected, `switchChain` will switch the target chain for the Wolf [`Config`](/react/api/createConfig#config).
:::
