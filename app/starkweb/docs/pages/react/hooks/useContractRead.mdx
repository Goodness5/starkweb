# useContractRead

Hook for calling a **read-only** function on a contract, and returning the response.

A **read-only** function (constant function) on a Cairo contract is denoted by a pure or view keyword. They can only read the state of the contract, and cannot make any changes to it. Since read-only methods do not change the state of the contract, they do not require any gas to be executed, and can be called by any user without the need to pay for gas.

## Import

```ts
import { useContractRead } from "sn-wolf"
```

## Usage

:::code-group
```tsx [index.tsx]
import { useContractRead } from "sn-wolf"
import { abi } from "./abi"

function App() {
  const result = useContractRead({
    abi,
    address: "0x06707675cd7dd9256667eca8284e46f4546711ee0054bc2dd02f0ce572056cf4",
    functionName: "totalSupply",
  })
}
```
